Azure Fundamentals

Course Information

  • Duration: 5 days
  • Schedule: Mon - Fri, 9:00 AM - 5:00 PM
  • Location: Online / On-site
  • Target Audience: Working Professionals, IT Staff, Developers

Instructor

John Smith, Cloud Architect with 12 years of experience in Azure and enterprise cloud solutions.

Course Level

Beginner / Professional Upskilling

Course Overview

This intensive 5-day course provides working professionals with a solid foundation in Microsoft Azure. Participants will gain hands-on experience with core Azure services, cloud concepts, and best practices, enabling them to confidently leverage Azure for business solutions.

Learning Objectives

  • Understand cloud computing concepts and Azure architecture
  • Navigate Azure Portal and manage resources
  • Implement core Azure services including VMs, Storage, and Networking
  • Explore security, compliance, and governance in Azure
  • Prepare for Azure Fundamentals certification (AZ-900)

5-Day Schedule

Day Topic Activity / Hands-On
Day 1 Introduction to Cloud & Azure Lecture + Azure Portal Exploration
Day 2 Core Azure Services: Compute & Storage Hands-on Lab: Deploy VMs, Storage Accounts
Day 3 Networking & Identity Hands-on Lab: VNets, Azure AD, Role-based Access
Day 4 Security, Compliance & Governance Lecture + Scenario Workshop
Day 5 Certification Prep & Capstone Project Project Presentation + Q&A
